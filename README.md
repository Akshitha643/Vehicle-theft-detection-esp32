# Vehicle-theft-detection-esp32
Vehicle Theft Detection and Protection System using ESP32

Technologies used:
- ESP32
- SIM800L GSM Module
- MPU6050 Accelerometer
- Keypad
- Buzzer

Features:
- Theft Detection (Tilt + Vibration + Movement)
- Accident Detection (Impact)
- SMS Alerts using GSM
- Buzzer Alarm
  
Applications:
- Vehicle security systems
- Accident alert systems
- Smart transportation
